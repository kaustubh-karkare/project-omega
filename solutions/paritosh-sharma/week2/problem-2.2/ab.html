It is always my pleasure to give the best i can and help people in getting just the right thing they have in mind.

What I Can Do


Development


I can work efficiently in a team as well as individually to make responsive websites. I can help you build a fully working website shaped according to your requirements. If required i can also handle basic backend-tasks.


Design

Beat Box

Beat Box is a fun game designed to test one's reflexes. It is a single page html5 game built using Bootstrap and Jquery and is mobile compatible.
